# Number System

This folder contains basic number-related problems implemented in Java.

## Problems Solved

| #  | Problem | Concept Used |
|--|--------|-------------|
| 1 | Prime Number | Loop, Square Root Optimization |
| 2 | Factorial | Recursive, Iterative |
| 3 | Palindrome Number | Reverse no using loops, modulo, division |
| 4 | Reverse Digits | Modulo (%) and division (/) with loop |
| 5 | GCD of two nos. | Euclidean Algorithm using modulo (%) and loop |
| 6 | LCM of two nos. | Use relation LCM(a,b) = (a*b) / GCD(a,b) |
